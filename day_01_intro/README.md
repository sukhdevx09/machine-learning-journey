## 🧠 What is Machine Learning?

Machine Learning is a field of AI where systems learn patterns from data and make predictions or decisions without being explicitly programmed.

---

## 📚 Types of Machine Learning

### 1. Supervised Learning
- Uses labeled data
- Example: predicting house prices

### 2. Unsupervised Learning
- Finds patterns in unlabeled data
- Example: clustering customers

### 3. Reinforcement Learning
- Learns through rewards and penalties
- Example: game playing AI

---

## 🔑 Key Concepts

- Data is the foundation of ML  
- Models learn patterns from data  
- Performance depends on quality of data
